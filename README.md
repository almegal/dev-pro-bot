Проект "Чат-бот для приюта"

Цель проекта
Цель нашего проекта - создать чат-бота для приюта, который будет помогать посетителям узнавать информацию о приюте, а также помогать в усыновлении животных.

Участники команда №3
Aleksey - разработчик
Исмаил  - разработчик
Алкесей - разработчик

 Задачи проекта
1. Предоставить информацию о приюте и его животных.
2. Помочь потенциальным усыновителям найти подходящего питомца.
3. Собирать отчеты от усыновителей о новых питомцах.
